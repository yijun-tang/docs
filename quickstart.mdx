## Prerequisites

Before you begin, make sure you have:

- Installed Selected app for your desktop
- Signed in with a free plan or paid plan (Plus or Pro)

## Fix grammar example

Learn how to use Selected's built-in commands to fix spelling and grammar in any application.

### Step 1: Select text

Select some raw text in any application on your desktop.

### Step 2: Open the popup

Hit the shortcut to display the popup:

- **macOS**: `Option + Space`
- **Windows/Linux**: `Ctrl + Space`

### Step 3: Choose a command

1. Type `/fix` to filter built-in commands with the prefix `fix`
2. Choose **Fix spelling & grammar** using the Enter key or mouse
3. Hit Enter or click the execute button to run the command

### Step 4: Review and apply

1. Wait for the corrected version to appear
2. If you're satisfied with the result, use one of the following built-in commands:
   - **Insert back**: Replace the original text with the corrected version
   - **Copy to clipboard**: Copy the corrected text for use elsewhere

## Learn More

The **Fix spelling & grammar** command is just one of many built-in commands available in Selected. To learn more about all available commands, use the **Help** command within the app, or explore the functionality documentation pages.
