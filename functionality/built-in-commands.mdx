## Customizable Commands

| Command                    | Description                                                           | Selected Text | Attachment | Custom Prompt |
| -------------------------- | --------------------------------------------------------------------- | ------------- | ---------- | ------------- |
| **Improve writing**        | Improves wording, sentence flow, and clarity without changing meaning | required      | ignored    | ignored       |
| **Fix spelling & grammar** | Corrects spelling, grammar, and punctuation errors                    | required      | ignored    | ignored       |
| **Translate**              | Translates text to various languages                                  | required      | ignored    | ignored       |
| **Make shorter**           | Reduces wordiness while keeping essential meaning                     | required      | ignored    | ignored       |
| **Simplify writing**       | Rewrites in simpler language without jargon                           | required      | ignored    | ignored       |
| **Change tone**            | Rewrites text in a different tone                                     | required      | ignored    | ignored       |
| **Make longer**            | Expands ideas with supporting details and examples                    | required      | ignored    | ignored       |
| **Continue writing**       | Continues from where the text stops, matching style                   | required      | ignored    | ignored       |
| **Summarize**              | Produces focused summaries highlighting key points                    | required      | ignored    | ignored       |
| **List key takeaways**     | Extracts actionable insights as bullet points                         | required      | ignored    | ignored       |
| **Find action items**      | Identifies concrete tasks or next steps                               | required      | ignored    | ignored       |
| **Explain**                | Breaks down complex ideas into easy explanations                      | required      | ignored    | ignored       |
| **Slack reply**            | Drafts professional and friendly Slack message replies                | required      | optional   | optional      |
| **Email reply**            | Drafts professional email replies with proper formatting              | required      | optional   | optional      |

## Non-Customizable Commands

| Command               | Description                                                      | Selected Text | Attachment | Custom Prompt |
| --------------------- | ---------------------------------------------------------------- | ------------- | ---------- | ------------- |
| **Help**              | Displays help information and guidance on using Selected         | ignored       | ignored    | ignored       |
| **Attachment**        | Add selected text as attachment for providing additional context | required      | ignored    | ignored       |
| **Insert back**       | Inserts the generated response back into original app            | ignored       | ignored    | ignored       |
| **Copy to clipboard** | Copies the response to your clipboard                            | ignored       | ignored    | ignored       |
| **Regenerate**        | Regenerates the last response with the same inputs               | ignored       | ignored    | ignored       |

## Customization

You can customize the prompts for the 14 writing task commands listed above to better fit your needs. Built-in commands customization is similar to custom commands creation, please refer to the [Custom Commands](/functionality/custom-commands) page.
